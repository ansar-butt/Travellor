Travellor:
Travellor serves as a proof of concept for Hotel Booking apps for Pakistan. The User Interface is easy to Manipulate and various features are avaliable to the users, such as viewing hotels from various regions, filtering hotels according to user needs, booking, unbooking and more.

Technologies:
We have used a Java based implementation, with Firebase serving as our database. Furthermore, Google AdMob is used for app monetisation although it is currently implemented with a test-key.
